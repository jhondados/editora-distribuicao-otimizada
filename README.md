# 📚 Otimização de Distribuição

> Otimização da distribuição de livros para livrarias, escolas e CDs. Reduz custo logístico e tempo de entrega, equilibrando estoque entre pontos com previsão de demanda.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/editora-distribuicao-otimizada/

## 📊 Resultados-chave (KPIs)
**-22%** Custo · **+18%** Prazo · **balanceado** Estoque · **4.200** Pontos

## 🧠 Tecnologias de IA
Optimization · ML · Supply Chain

## ✨ O que faz
- Roteirização de distribuição
- Balanceamento de estoque
- Previsão de demanda por ponto
- Redução de custo logístico
- Nível de serviço por região

## 🗄️ Fonte de dados real
[Rede de distribuição + vendas](https://www.cbl.org.br)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com Rede de distribuição + vendas
2. **Processamento com IA** — Optimization · ML · Supply Chain
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
